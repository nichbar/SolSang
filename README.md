# Sol Sang
一款轻松使用颜文字的APP.

支持Android 4.2(SDKversion > 17)以上设备，算是一个练手用的APP。

只需要一个权限“向用户词典添加字词”。

采用Material Design设计，比较耐看，•ิ_• ิ。

###应用截图
![img](http://ww1.sinaimg.cn/large/62cc645cgw1eyaq0f9f3qj20dc0o0jsc.jpg)

###使用说明
点击颜文字复制到剪贴板

长按弹出选项框
